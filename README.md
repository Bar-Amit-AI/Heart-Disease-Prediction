# Heart Disease Classification

## Overview

This project analyzes cardiovascular health data and applies machine learning classification models to predict heart disease.

Originally developed as part of the Magshimim AI program and later reorganized into a standalone portfolio project.

## Features

- Data cleaning
- Exploratory Data Analysis (EDA)
- Feature engineering
- Model training
- Model evaluation

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
